# دليل نشر تطبيق Shego and Hego على Google Play

## المتطلبات الأساسية

### 1. حساب Google Play Developer
- **الرسوم**: $25 (مرة واحدة)
- **الرابط**: https://play.google.com/console
- **المتطلبات**:
  - حساب Google نشط
  - طريقة دفع صحيحة
  - معلومات الحساب البنكي

### 2. شهادة التوقيع (Signing Certificate)

#### إنشاء مفتاح التوقيع
```bash
# إنشاء keystore جديد
keytool -genkey -v -keystore shego-and-hego.keystore \
  -keyalg RSA -keysize 2048 -validity 10000 \
  -alias shego-and-hego-key

# ستُطلب منك المعلومات التالية:
# - كلمة المرور (يجب حفظها)
# - الاسم الأول والأخير
# - اسم المؤسسة
# - اسم المدينة
# - اسم الولاية
# - رمز الدولة (مثل: US, SA, EG)
```

#### معلومات التوقيع
- **اسم الملف**: shego-and-hego.keystore
- **كلمة المرور**: [احفظ كلمة المرور بأمان]
- **الصلاحية**: 10 سنوات

#### بناء APK
```bash
# تثبيت EAS CLI
npm install -g eas-cli

# تسجيل الدخول
eas login

# بناء APK
eas build --platform android --type apk

# أو بناء AAB (الموصى به)
eas build --platform android --type app-bundle
```

### الخطوة 2: إنشاء قائمة التطبيق في Google Play Console

1. انتقل إلى https://play.google.com/console
2. اضغط على "إنشاء تطبيق"
3. أدخل اسم التطبيق: "Shego and Hego"
4. حدد الفئة: "Social"
5. اختر نوع التطبيق: "تطبيق"

### الخطوة 4: إضافة الصور والفيديوهات

#### الصور المطلوبة
1. **أيقونة التطبيق**: 512×512 بكسل (PNG)
2. **صور لقطات الشاشة**: 5-8 صور (1080×1920 بكسل)
3. **صورة الميزة**: 1024×500 بكسل (PNG/JPG)
4. **صورة العنوان**: 512×512 بكسل (PNG)

#### لقطات الشاشة الموصى بها
```
1. الشاشة الرئيسية - عرض الفيديوهات
2. شاشة التحميل - رفع الفيديوهات
3. شاشة الملف الشخصي - إحصائيات المستخدم
4. شاشة الرسائل - التواصل
5. شاشة الأرباح - كسب الأموال
6. شاشة الإعدادات - تخصيص التطبيق
7. شاشة المتابعة - متابعة المستخدمين
8. شاشة الهدايا - نظام الهدايا
```

### الخطوة 7: رفع ملف APK/AAB

1. انتقل إلى "الإصدارات" → "الإنتاج"
2. اضغط على "إنشاء إصدار جديد"
3. حمّل ملف APK أو AAB
4. أضف ملاحظات الإصدار:
```
الإصدار 1.0.0 - الإطلاق الأولي

الميزات الجديدة:
- تطبيق جديد تماماً
- دعم 50+ لغة
- نظام دفع متكامل
- إعلانات محسّنة
- أداء محسّن
```

### الخطوة 8: الموافقة على الإعلانات والمحتوى

- تأكد من أن التطبيق يتبع سياسات Google Play
- لا توجد محتويات محظورة
- لا توجد انتهاكات للملكية الفكرية

### الخطوة 9: الاختبار

قبل النشر، اختبر التطبيق على:
- أجهزة مختلفة
- إصدارات Android مختلفة
- اتصالات إنترنت مختلفة
- الأداء والاستقرار

### الخطوة 10: النشر

1. راجع جميع المعلومات
2. اضغط على "مراجعة"
3. اضغط على "نشر"
4. انتظر الموافقة (عادة 1-3 ساعات)
